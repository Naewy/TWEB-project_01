What is GitHub

GitHub is a web-based Git or version control repository and Internet hosting service. It is mostly used for code. It offers all of the distributed version control and source code management functionality of Git as well as adding its own features. It provides access control and several collaboration features such as bug tracking, feature requests, task management, and wikis for every project

How does it work

This website is designed to show the number of issues that are currently in an open state for a randomly chosen enterprise. Every day, a new one is selected and the top 10 repositories are listed in the table and the graph bellow. Today's organisation is . All the data are gathered via the GitHub API and the results are stored on our GitHub repository for later use.
